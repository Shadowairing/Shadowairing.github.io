第二步，修改BIOS

开机按F2进入BIOS设置

  • Configuration
    • System Performance Mode: Intelligent 解决20%以下电量时蓝牙关闭问题
  • Security
    • Intel SGX: Disable or Software Control 视BIOS版本不同操作会有所不同
  • Exit
    • OS Optimized Defaults: Disabled

联想小新Air13IWL隐藏BIOS推荐设置

  • 隐藏BIOS进入姿势
    • 电源键开机 → F2进入正常BIOS → 电源键关机 → 然后顺序按下下列键
      • F1 → 1 → Q → A → Z
      • F2 → 2 → W → S → X
      • F3 → 3 → E → D → C
      • F4 → 4 → R → F → V
      • F5 → 5 → T → G → B
      • F6 → 6 → Y → H → N
    • 电源键开机 → F2进入隐藏BIOS , 如不成功请加快手速再次尝试
  • 推荐设置选项
    • Advanced → Power & Performance → CPU - Power Management Control → CPU Lock Configuration → CFG Lock → Disabled
    • Advanced → System Agent (SA) Configuration → Graphics Configuration → DVMT Pre-Allocated → 64M
    • Advanced → PCH-IO Configuration → Seriallo Configuration → I2C1 Controller → Disabled

安装ALCPlugFix

简介

  • 它可以解决耳机插拔状态的切换。
  • 它是通过使用命令 : hda-verb 0xNode SET_PIN_WIDGET_CONTROL 0xVerbs 的方式进行状态切换
  • hda-verb 是原本来自于linux下面的 alsa-project 的一条命令 , 它的作用是发送HD-audio命令

注意：

  • 每次开机和睡眠唤醒后 , 你可能需要重新插拔耳机来让耳机正常工作
  • 安装前需要确认 SIP 为关闭状态 , 使用本仓库的EFI,默认为关闭状态
    • macOS 10.15 Catalina 需要临时解锁系统分区 , 终端执行以下命令后再安装
      • 挂载为可写模式 : sudo mount -uw /
      • 重启 Finder : killall Finder

根据需要安装HIDPI，安装之后盒盖开盖之后显示可能会不正常，但是不影响使用

简介

macOS 的 DPI 机制和 Windows 下不一样 , 比如 1080P 的屏幕在 Windows 下有 125% 、150% 这样的缩放选项 , 而同样的屏幕在 macOS 下 , 缩放选项里只是单纯的调节分辨率 , 这就使得在默认分辨率下字体和UI看起来很小 , 降低分辨率又显得模糊 , 此脚本的目的是为开启 HiDPI 选项 , 使屏幕具有原生的 HIDPI 设置 , 不需要 RDM 软件即可在系统显示器设置中设置 , 开机的第二阶段 logo 总是会稍微放大 , 因为分辨率是仿冒的

使用方法

  • 使用前需要确认 SIP 为关闭状态 , 使用本仓库的EFI,默认为关闭状态
    • macOS 10.15 Catalina 需要临时解锁系统分区 , 终端执行以下命令后再使用
      • 挂载为可写模式 : sudo mount -uw /
      • 重启 Finder : killall Finder
  • 在终端执行HIDPI目录中的 hidpi-zh.sh
  • 选择MacBook Pro
  • 选择1080P 显示屏
  • 重启

恢复

  • 如果使用此脚本后，开机无法进入系统，请到恢复模式中或使用 clover -x 安全模式进入系统 ，使用终端删除 /System/Library/Displays/Contents/Resources/Overrides 下的 DisplayVendorID-xxx 文件夹(xxx是显示器供应商的ID)，并把 backup 文件夹中的备份复制出来。

CPU 变频数据定制

上面EFI中提供的是Intel Core i5-8265U的驱动，我的CPU是Intel Core i7-8565U，所以可以定制一下（不定制也能用）

原本驱动是i5-8265U睿频到3.9Ghz，定制之后可以让i7-8565U睿频到4.6Ghz

  • 终端运行CPUFrequency目录中的View.sh
  • 输入密码,选nano继续,记下高亮绿色的机型
  • 在终端执行下面的命令，注意，将Mac-XXXXXXXXXXXXXXXX.plist替换成你的型号
  • 比如说我的机型是Mac-1E7E29AD0135F9BC.plist，就执行下面的命令
  • 修改桌面CPU.plist最低频率为800Mhz，将圈住的0D000000改为08000000
  • 执行CPUFrequency目录中的Data.sh，将会在桌面得到CPUFriendDataProvider.kext
  • 备份原来的EFI，将CPUFriendDataProvider.kext放进EFI里面的驱动文件夹里替换掉之前的，重启
    • 第一次可能不能制作成功，所以要先备份，以免翻车，我做了两次就成功了

关闭软件更新

在系统偏好设置里关闭软件更新，如果以后弹出软件更新小红点提示，用下面的方法去掉

在终端输入: defaults write com.apple.systempreferences AttentionPrefBundleIDs 0
然后输入: killall Dock
